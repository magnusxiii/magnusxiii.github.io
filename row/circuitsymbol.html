Σύμβολα Ηλεκτρονικών Κυκλωμάτων

Σύμβολο Περιγραφή Συνδέσεις
VCCΚύρια θετική τάση τροφοδοσίας (συνήθως για λογικά κυκλώματα TTL/CMOS, τρανζίστορ NPN).Συνδέεις την κύρια θετική τάση (π.χ. +3.3V, +5V, +12V).
VSSΣυνήθως γείωση ή αρνητική τάση σε CMOS/ψηφιακά κυκλώματα (Source Voltage).Συνδέεις GND ή αρνητική τάση, ανάλογα το κύκλωμα.
VDDΘετική τάση τροφοδοσίας για MOSFET/CMOS (Drain Voltage).Συνδέεις κύρια τάση τροφοδοσίας (συχνά ίδια με VCC).
V+Θετική τάση τροφοδοσίας σε διπολικά κυκλώματα (π.χ. op-amp).Συνδέεις θετική τάση (π.χ. +12V, +15V).
V-Αρνητική τάση τροφοδοσίας σε διπολικά κυκλώματα.Συνδέεις αρνητική τάση (π.χ. -12V, -15V).
VINΕίσοδος τάσης (Input Voltage).Συνδέεις την τάση εισόδου (π.χ. από τροφοδοτικό, μπαταρία).
VOUTΈξοδος τάσης (Output Voltage).Εδώ παίρνεις την τάση εξόδου (π.χ. από ρυθμιστή τάσης, ενισχυτή).
GNDΓείωση (Ground), σημείο αναφοράς.Συνδέεις το κοινό σημείο επιστροφής ρεύματος, αρνητικό τροφοδοσίας.
AGNDΑναλογική γείωση (Analog Ground).Συνδέεις γείωση για αναλογικά σήματα, ξεχωριστή από ψηφιακή για αποφυγή θορύβου.
DGNDΨηφιακή γείωση (Digital Ground).Συνδέεις γείωση για ψηφιακά κυκλώματα.
PGNDΓείωση ισχύος (Power Ground).Συνδέεις γείωση για κυκλώματα ισχύος (π.χ. MOSFETs, μετατροπείς DC/DC).
ENΕνεργοποίηση/Απενεργοποίηση κυκλώματος (Enable).Συνδέεις ψηφιακό σήμα (0/1 ή LOW/HIGH) για ενεργοποίηση ή απενεργοποίηση.
ENABLΕναλλακτική ονομασία για Enable.Συνδέεις ψηφιακό σήμα για ενεργοποίηση/απενεργοποίηση.
ENABLEΕναλλακτική ονομασία για Enable.Συνδέεις ψηφιακό σήμα για ενεργοποίηση/απενεργοποίηση.
CTRLΈλεγχος λειτουργίας (Control).Συνδέεις σήμα ελέγχου (αναλογικό ή ψηφιακό).
CNTRLΕναλλακτική ονομασία για Control.Συνδέεις σήμα ελέγχου.
DTCDead Time Control, ρύθμιση νεκρού χρόνου σε κυκλώματα PWM.Συνδέεις αντίσταση/πυκνωτή για ρύθμιση dead time.
SSSoft Start, ομαλή εκκίνηση τροφοδοσίας.Συνδέεις πυκνωτή για καθυστέρηση στην εκκίνηση.
CTTiming Capacitor, πυκνωτής χρονισμού.Συνδέεις πυκνωτή για χρονισμό (σε ταλαντωτές, PWM, timers).
SYNCΣυγχρονισμός (Synchronization).Συνδέεις σήμα συγχρονισμού από άλλο κύκλωμα ή γεννήτρια.
RESETΕπαναφορά (Reset), αρχικοποίηση κυκλώματος.Συνδέεις σήμα reset (συνήθως μέσω κουμπιού ή κυκλώματος watchdog).
FBFeedback, ανατροφοδότηση.Συνδέεις σήμα από έξοδο προς είσοδο ελέγχου (π.χ. σε ρυθμιστές τάσης, ενισχυτές).
SENSEΜέτρηση ρεύματος ή τάσης (Sense).Συνδέεις σημείο μέτρησης (π.χ. μετά από shunt resistor).
SNSΕναλλακτική ονομασία για Sense.Συνδέεις σημείο μέτρησης.
ISENCurrent Sense, μέτρηση ρεύματος.Συνδέεις shunt resistor ή κύκλωμα μέτρησης ρεύματος.
CSCurrent Sense ή Chip Select (ανάλογα το κύκλωμα).Συνδέεις shunt resistor για μέτρηση ρεύματος ή σήμα επιλογής chip σε SPI.
SHUNTShunt Resistor, αντίσταση μέτρησης ρεύματος.Συνδέεις χαμηλής τιμής αντίσταση σε σειρά με το φορτίο.
PGOODPower Good, ένδειξη σωστής λειτουργίας τροφοδοσίας.Συνδέεις LED, μικροελεγκτή ή κύκλωμα παρακολούθησης.
FAULTΈνδειξη σφάλματος.Συνδέεις κύκλωμα ειδοποίησης ή μικροελεγκτή για ανίχνευση σφαλμάτων.
OVPOver Voltage Protection, προστασία υπέρτασης.Συνδέεις κύκλωμα προστασίας ή παρακολούθησης.
UVLOUnder Voltage Lock Out, προστασία υποτάσεως.Συνδέεις κύκλωμα παρακολούθησης τάσης.
CLKΡολόι (Clock), σήμα χρονισμού.Συνδέεις γεννήτρια ρολογιού ή εξωτερικό oscillator.
XTALΚρύσταλλος (Crystal Oscillator).Συνδέεις κρύσταλλο χαλαζία για ακριβή χρονισμό.
OSCOscillator, ταλαντωτής.Συνδέεις εξωτερικό ταλαντωτή ή κύκλωμα χρονισμού.
INΕίσοδος σήματος.Συνδέεις σήμα εισόδου (αναλογικό ή ψηφιακό).
OUTΈξοδος σήματος.Συνδέεις το φορτίο ή το επόμενο στάδιο κυκλώματος.
INTΔιακοπή (Interrupt).Συνδέεις σήμα διακοπής σε μικροελεγκτή ή CPU.
IOΕίσοδος/Έξοδος γενικής χρήσης (Input/Output).Συνδέεις σήματα εισόδου/εξόδου, κουμπιά, LED, αισθητήρες.
GPIOGeneral Purpose Input/Output, είσοδος/έξοδος γενικής χρήσης.Συνδέεις σήματα εισόδου/εξόδου, κουμπιά, LED, αισθητήρες.
TXΜετάδοση (Transmit), έξοδος σειριακής επικοινωνίας.Συνδέεις στη γραμμή RX άλλης συσκευής (π.χ. UART, RS232).
RXΛήψη (Receive), είσοδος σειριακής επικοινωνίας.Συνδέεις στη γραμμή TX άλλης συσκευής.
SDAData Line για I2C.Συνδέεις στη γραμμή δεδομένων I2C bus (pull-up resistor απαιτείται).
SCLClock Line για I2C.Συνδέεις στη γραμμή ρολογιού I2C bus (pull-up resistor απαιτείται).
MISOMaster In Slave Out (SPI).Συνδέεις έξοδο slave στη γραμμή εισόδου του master.
MOSIMaster Out Slave In (SPI).Συνδέεις έξοδο master στη γραμμή εισόδου του slave.
SCKSerial Clock (SPI).Συνδέεις σήμα ρολογιού SPI από master σε slave(s).
SSSlave Select (SPI).Συνδέεις σήμα επιλογής slave (ενεργοποίηση slave όταν είναι LOW).
SPISerial Peripheral Interface.Συνδέεις όλα τα παραπάνω (MISO, MOSI, SCK, SS) για SPI επικοινωνία.
BOOTBoot Mode, επιλογή λειτουργίας εκκίνησης.Συνδέεις σε GND ή VCC για επιλογή τρόπου εκκίνησης (π.χ. bootloader).
BSTBoost, pin για floating capacitor σε buck/boost converters.Συνδέεις πυκνωτή boost (bootstrap) μεταξύ BST και SW.
NCNo Connection, δεν συνδέεται.Δεν συνδέεις τίποτα (αφήνεται ανοιχτό).
EPADExposed Pad, μεταλλική βάση για ψύξη/γείωση.Συνδέεις στη γείωση PCB για θερμική διαχείριση και ηλεκτρική σύνδεση.
TEMPΘερμοκρασία, pin αισθητήρα θερμοκρασίας.Συνδέεις αισθητήρα ή κύκλωμα μέτρησης θερμοκρασίας.
ADDRΔιεύθυνση (Address), επιλογή διεύθυνσης σε bus (π.χ. I2C).Συνδέεις σε GND ή VCC για να ορίσεις μοναδική διεύθυνση.
RTTiming Resistor, αντίσταση χρονισμού.Συνδέεις αντίσταση για ρύθμιση συχνότητας ή χρόνου.
COMPCompensation, σταθεροποίηση ελέγχου (συνήθως σε ρυθμιστές τάσης).Συνδέεις κύκλωμα αντιστάσεων/πυκνωτών για σταθεροποίηση loop.
